# AMD P-State EPP Scaling Driver Script
Script to enable AMD P-State EPP Scaling Driver `amd-pstate-epp` for AMD CPUs that support it.

Documentation:
- https://github.com/Hekel1989/amdpstate-configuration?tab=readme-ov-file

## Requirements
- AMD processor that supports `amd-pstate-epp`. Most Zen2 and Zen3 processors are supported.
- CPPC support enabled in your UEFI. This option may not be present but still enabled by default.
